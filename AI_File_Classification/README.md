
# AI-Based File Classification System

This project automatically organizes randomly arranged files using Artificial Intelligence.

## Features
- Uses NLP & Machine Learning
- TF-IDF + Naive Bayes Classifier
- Automatically categorizes text files
- Beginner-friendly AI project

## Technologies Used
- Python
- Scikit-learn
- NLP
- Machine Learning

## How to Run
1. Install dependencies:
   pip install scikit-learn
2. Update folder path in main.py
3. Run:
   python main.py

## Output
Files are organized into folders like:
- Bills
- Study
- Office
- Finance
- Resumes
